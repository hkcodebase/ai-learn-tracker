# AI Engineer Prep Tracker

A 10-week daily study tracker for Software/Data Engineers specialising in
Gen AI — covering LLM fundamentals, RAG, agents, MCP, evals, and system design.

## What it covers

| Phase | Weeks | Focus |
|-------|-------|-------|
| Foundations | 1–2 | LLM basics, tokenization, quantization |
| RAG | 3–4 | Retrieval, chunking, guardrails, hands-on build |
| Agents | 5–6 | Tool calling, ReAct, MCP, multi-agent systems |
| Evals | 7 | LLM-as-judge, RAGAS, production AI practices |
| System Design | 8 | Agentic systems at scale, mock interviews |
| Capstone | 9–10 | Multimodal overview, portfolio project |

## Features

- Daily check-ins with topic, sub-topic, concepts, and resource links
- Per-week progress bars + overall completion tracker
- Notes field per day — saves automatically in your browser
- Print-ready layout
- No backend needed — progress saves in localStorage

## Deploy your own

1. Fork this repo
2. Go to Settings → Pages → Source: GitHub Actions
3. Push to `main` — auto-deploys via the included workflow
4. Optional: add a `CNAME` file with your custom domain

## Tech

Single static HTML file. No framework. No build step. No dependencies.
JetBrains Mono via Google Fonts. localStorage for persistence.
